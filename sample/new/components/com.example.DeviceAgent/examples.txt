Always use journalctl to check for the reason, you can check as many lines of log as you want.
Use single quotes (') instead of double quotes (\") for parameters with spaces to avoid JSON parsing issues.

Device Agent Command Examples:

System Information:
uname -a
uptime
whoami

Disk & Memory:
df -h
free -h
du -sh /var/lib/greengrass

Service Status:
systemctl status ggl.com.example.Example.service

Check for all ggl related Components on Device:
systemctl list-units --type=service --state=loaded | grep ggl

Check for all component on Device:
systemctl list-units --type=service --state=loaded

Log Commands (use single quotes for timestamps):
journalctl -u ggl.com.example.SystemMonitorC.service --lines=10 --no-pager
journalctl --since '1 hour ago' --no-pager -n 20
journalctl --priority=err --since '1 hour ago' --no-pager

Network:
ip addr show
ip route
